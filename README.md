# Data Warehouse and Analytics Project

## 1. Project Architecture & Framework

This project implements a modern 3-Layer Medallion Architecture using SQL Server to ingest, clean, and model operational business data. The pipeline handles data from two distinct corporate sources (ERP and CRM systems) and processes it into an analytics-ready warehouse.

**Status:** Active Development 🚀
* **Currently implementing:** Silver layer transformations.
* **Completed:** Bronze layer ingestion procedures.

## 2. Tech Stack

* **Database:** SQL Server
* **Methodology:** Medallion Architecture (Bronze, Silver, Gold)
* **Scope:** ETL Pipeline Development, Data Modeling, and Data Cleaning
